# O que é Selenium?

Uma biblioteca que controla um navegador real (Chrome, Firefox...) por código — clicando, digitando e navegando como um humano.

  • Testes automatizados de sites e sistemas web
  • Web scraping de páginas que dependem de JavaScript
  • Automação de tarefas repetitivas (logins, formulários, relatórios)

Se um humano consegue fazer no navegador, o Selenium consegue automatizar.

# Como funciona

  • Seu script Python fala com o WebDriver
  • O WebDriver controla um Chrome de verdade na tela
  • Selenium 4 baixa o driver sozinho — sem configuração

Você vê tudo acontecendo: o mouse, os cliques, as páginas trocando.

# abre o Chrome (driver automático)
navegador = webdriver.Chrome()
navegador.get("https://siga...")

# Os 2 conceitos-chave

1. Localizar — By

Acha elementos por ID, XPATH, classe...

2. Esperar — WebDriverWait

Páginas carregam devagar. O Selenium espera o elemento ficar pronto antes de agir.

# espera o campo aparecer, então digita
campo = espera.until(
    EC.element_to_be_clickable(
        (By.ID, "i0116")
    )
)
campo.send_keys(EMAIL)
